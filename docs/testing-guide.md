# AI4Edu 测试指南

## 快速开始

### 1. 启动开发服务器
```bash
pnpm dev
```

### 2. 打开浏览器访问
```
http://localhost:1420
```

---

## 功能测试清单

### ✅ 基础界面测试
- [ ] 页面正常加载，显示"AI4Edu 高中数学助手"
- [ ] 左侧边栏显示快捷操作和历史记录
- [ ] 主界面显示4个快捷入口卡片

### ✅ 智能出题功能测试
1. 点击 **"智能出题"** 卡片
2. **步骤1 - 选择知识点**：
   - [ ] 显示7个知识点选项（函数、导数、三角函数等）
   - [ ] 点击"导数"进入下一步
3. **步骤2 - 选择难度**：
   - [ ] 显示3个难度级别（基础/中档/困难）
   - [ ] 点击"中等"进入下一步
4. **步骤3 - 选择数量**：
   - [ ] 显示4个数量选项（3/5/8/10道）
   - [ ] 选择"5道"进入下一步
5. **步骤4 - 确认生成**：
   - [ ] 显示配置摘要
   - [ ] 点击"开始生成题目"
   - [ ] 等待1-2秒，看到生成的题目内容（含LaTeX公式）

### ✅ PPT讲评课测试
1. 点击 **"试卷讲评"** 卡片
2. **步骤1 - 上传试卷**：
   - [ ] 显示文件上传区域
   - [ ] 点击"跳过上传，手动选择"
3. **步骤2 - 选择讲评重点**：
   - [ ] 显示5个选项（高频错题、概念混淆等）
   - [ ] 勾选"高频错题"和"概念混淆"
   - [ ] 点击"下一步"
4. **步骤3 - 课件配置**：
   - [ ] 选择幻灯片数量（如12页）
   - [ ] 勾选"包含错题分析"和"包含补救练习"
   - [ ] 点击"下一步"
5. **步骤4 - 确认生成**：
   - [ ] 显示完整配置信息
   - [ ] 点击"生成讲评课件"

### ✅ 导出功能测试
1. 查看AI生成的回复消息
2. [ ] 消息底部显示"导出 Word"和"导出 PPT"按钮
3. 点击"导出 Word"：
   - [ ] 显示加载动画
   - [ ] 浏览器自动下载 .doc 文件

### ✅ 自由对话测试
1. 在输入框输入任意文字（如"你好"）
2. [ ] 按回车发送消息
3. [ ] AI回复欢迎消息
4. [ ] 消息底部显示3个快捷按钮（出题/做课件/改题目）

## 问题排查

### 页面空白
```bash
# 检查控制台错误
F12 → Console

# 重启服务器
pkill -f vite
pnpm dev
```

### 样式异常
- 强制刷新：Ctrl+F5
- 检查网络标签，确认CSS已加载

### 功能无响应
1. 打开浏览器开发者工具（F12）
2. 查看Console是否有红色错误
3. 检查Network标签是否有API请求失败
