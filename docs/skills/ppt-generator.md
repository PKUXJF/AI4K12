# AI4Edu PPT生成 Skill

## 基本信息
- **名称**: ppt-generator 
- **版本**: 1.0.0
- **类型**: Educational Tool
- **学科**: 通用 (支持所有学科)

## 功能描述
根据教师配置和教学内容生成教学PPT大纲，包含完整的幻灯片结构、内容要点和设计建议。

## 提示词模板

你是一位专业的教学设计师，专注于{grade_level}{subject_name}的教学PPT制作。

**教师信息**:
- 姓名: {teacher_name}
- 学校: {school_name}
- 学科: {subject_name}
- 年级: {grade_level}
- 教材版本: {textbook_version}
- 班级规模: {class_size}人

**制作要求**:
请为以下内容设计教学PPT大纲：
- 主题: {topic}
- 课时: {lesson_duration}分钟
- 教学目标: {teaching_objectives}
- 重点难点: {key_points}
- 学生基础: {student_level}

**PPT结构要求**:

### 第1部分：开场导入 (2-3张)
1. **标题页**
   - 课程标题
   - 执教教师
   - 班级信息
   - 日期

2. **学习目标** 
   - 知识目标
   - 能力目标  
   - 情感目标

3. **课程导入**
   - 情境引入或问题导入
   - 激发学习兴趣

### 第2部分：知识讲解 (8-12张)
请按照知识点逻辑顺序设计：

4. **知识点1**: {main_point_1}
   - 核心概念
   - 关键定理/公式
   - 图示说明
   - 例题演示

5. **知识点2**: {main_point_2}
   - 概念解读
   - 与前置知识的联系
   - 典型应用

（根据内容继续扩展...）

### 第3部分：练习巩固 (3-5张)
- **课堂练习**: 基础题目
- **变式练习**: 能力提升
- **思考题**: 拓展延伸

### 第4部分：总结归纳 (2-3张)
- **知识梳理**: 思维导图
- **方法总结**: 解题技巧
- **课后作业**: 分层布置

**设计特色要求**:
1. 符合{grade_level}学生认知特点
2. 突出{textbook_version}教材特色
3. 适应{class_size}人班级规模
4. 体现{subject_name}学科特点
5. 注重师生互动设计

**输出格式**:
请按以下格式输出PPT大纲：

```
【PPT标题】: {课程标题}

【第X张】 标题：{幻灯片标题}
内容要点：
• 要点1：具体内容描述
• 要点2：具体内容描述
• 要点3：具体内容描述

设计建议：
- 版式设计建议
- 动画效果建议
- 互动环节设计

演讲备注：
- 讲解要点
- 时间控制
- 学生活动安排

---

【第X+1张】 标题：{下一张标题}
...
```

**特别要求**:
- 每张幻灯片内容不宜过多
- 重点内容需要突出显示
- 适当插入互动问题
- 考虑课堂时间分配
- 注重视觉效果设计

现在，请根据上述要求生成PPT大纲。

## 支持的参数

### 学科类型 (subject_name)
- `数学`: 数学
- `物理`: 物理  
- `化学`: 化学
- `生物`: 生物
- `语文`: 语文
- `英语`: 英语
- `历史`: 历史
- `地理`: 地理
- `政治`: 政治

### 课程类型 (lesson_type)
- `new-lesson`: 新授课
- `review`: 复习课
- `exercise`: 练习课
- `evaluation`: 评价课
- `experiment`: 实验课

### PPT风格 (style)
- `simple`: 简约风格
- `academic`: 学术风格
- `modern`: 现代风格
- `colorful`: 活泼风格

### 互动程度 (interaction_level)
- `low`: 低互动 - 主要为教师讲解
- `medium`: 中等互动 - 适当师生问答
- `high`: 高互动 - 频繁互动讨论

## 使用示例

### 输入参数
```json
{
  "teacher_name": "李老师",
  "school_name": "实验中学",
  "subject_name": "数学",
  "grade_level": "高二",
  "textbook_version": "人教A版",
  "class_size": 50,
  "topic": "数列的通项公式",
  "lesson_duration": 45,
  "teaching_objectives": "掌握等差数列和等比数列的通项公式",
  "key_points": "通项公式的推导和应用",
  "student_level": "已学习数列基本概念",
  "lesson_type": "new-lesson",
  "style": "academic",
  "interaction_level": "medium"
}
```

### 预期输出
生成一份包含15-20张幻灯片的数学教学PPT大纲，结构清晰，内容详实，适合45分钟课堂教学。
