## 简介

AI4Edu 是一款专为高中数学教师和学生设计的 AI 桌面助手。基于 Accomplish（MIT 开源）构建，支持智能出题、题目解析、教案生成等功能。

## ✨ 核心功能

- 📝 **智能出题** - 根据知识点、难度自动生成高质量数学题
- 📚 **题目解析** - AI 自动提供详细解答和多种解法
- 📖 **教案生成** - 一键生成教学大纲和课堂讲义
- 🎯 **LaTeX 支持** - 完美渲染数学公式
- 🔒 **本地优先** - 数据保存在本地，保护隐私

## 🚀 快速开始

### 配置 AI 服务

1. 打开设置 → AI 服务
2. 选择 **OpenAI**（SiliconFlow 兼容）
3. 填入配置：
   - API 地址：`https://api.siliconflow.cn/v1`
   - API Key：你的 SiliconFlow API Key
   - 模型：`Pro/moonshotai/Kimi-K2.5`
